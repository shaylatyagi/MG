# MobilityGrid Engine 

A fleet management platform for EV and commercial vehicle rental businesses in India.

## What it does

- Drivers pay daily rent via UPI
- Owners track collections, assign vehicles, manage drivers
- Admin monitors the full platform

## Live

[mg-sandy.vercel.app](https://mg-sandy.vercel.app)

## Stack

React · Node.js · PostgreSQL · PayYantra · Vercel · Render

## Structure

```
frontend/   → React app (Vercel)
backend/    → Express API (Render)
```

## Users

| Role | Access |
|------|--------|
| Admin | Full platform view |
| Owner | Their fleet & drivers |
| Driver | Payments & wallet |
| Manager | Owner-defined permissions (Premium) |
